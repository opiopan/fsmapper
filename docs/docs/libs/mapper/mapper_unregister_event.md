# mapper.unregister_event()
```lua
mapper.unregister_event(event_id)
```
This function unregisters an event.


## Prameters
|Parameter|Type|Description|
|-|-|-|
|`event_id`|numeric|Event ID of the event|


## Return Values
This function doesn't return any value.
