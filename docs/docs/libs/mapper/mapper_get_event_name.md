# mapper.get_event_name()
```lua
mapper.get_event_name(evet_id)
```
This function returns the name assinged to an event.


## Parameters
|Parameter|Type|Description|
|-|-|-|
|`evet_id`|number|Event ID|


## Return Values
This function returns the name assinged to an event.
