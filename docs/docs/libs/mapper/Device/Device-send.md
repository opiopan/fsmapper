# Device:send()
```lua
Device:send(unit_id, value)
```
This method sends a value to output unit.


## Prameters
|Parameter|Type|Description|
|-|-|-|
|`unit_id`|numeric|Output device unit ID|
|`value`|numeric|Value to be sent|


## Return Values
This method doesn't return any value.
