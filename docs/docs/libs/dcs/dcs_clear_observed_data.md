# dcs.clear_observed_data()
```lua
dcs.clear_observed_data()
```
Clear all DCS World cockpit data to be observed registered by [`dcs.add_observed_data()`](/libs/dcs/dcs_add_observed_data).


## Return Values
This function doesn't return any value.
