# dcs.clear_chunk()
```lua
dcs.clear_chunk()
```
This function clear all Lua chunks registered for execution within the DCS World process using [`dcs.register_chunk()`](/libs/dcs/dcs_register_chunk).


## Return Values
This function doesn't return any value.
